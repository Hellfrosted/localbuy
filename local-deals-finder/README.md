# Local Deals Finder

Search multiple used goods marketplaces at once.

![Screenshot](screenshot.png)

## Features

- Search 7 platforms simultaneously (Facebook Marketplace, Craigslist, OfferUp, Mercari, GovDeals, eBay, Nextdoor)
- Distance radius filter (5-100 miles)
- Save favorite searches
- Recent search history

## Usage

1. Open `index.html` in your browser
2. Allow pop-ups when prompted
3. Enter ZIP code and search term
4. Click **Search All Platforms**

## Requirements

- Modern web browser (Chrome, Firefox, Edge)
- Pop-ups must be allowed for multi-tab search
